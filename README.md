# Hinata Installer.

Replace original "safe.rpx" for custom rpx binary. 

Requires FailST installed on Health and Safety Information channel.

## Build

Run `make` in arm_user, arm_kernel and wupserver and copy each `*bin.h` to /payload  
Run `make`
